# Smart Greenhouse Monitoring System 

IoT-based system for real-time monitoring of environmental conditions inside a greenhouse using ESP32.

## Overview
This project collects temperature, humidity, and gas data using sensors connected to ESP32 and sends it to the ThingSpeak cloud for real-time visualization and analysis.

## Features
- Real-time data collection using sensors
- Cloud integration using ThingSpeak (HTTP over Wi-Fi)
- Data visualization through dashboards
- Local display using LCD
- Full system simulation using Wokwi

## Technologies
- ESP32
- ThingSpeak
- HTTP Protocol
- Wokwi Simulation
